# ❄️ 퀵숙 Quick Sook

| 출시일   | 버전   |
| -------- | ------ |
| 2020.09.17| v1.0.0 |

**웨일브라우저**를 사용하는 **숙명인**들을 위해 확장앱을 만들었습니다.
[다운 받기](https://store.whale.naver.com/detail/lniagbhflokdpfoilcmcpdmaphllolai)

## 주요 기능

숙명인들이 자주 사용하는 웹사이트를 모아두었습니다.

총 9개의 사이트를 제공하며 제공 범위는 다음과 같습니다.

- 숙명여자대학교 홈페이지
- 숙명 포털
- 스노위
- 스노보드
- 교내 번호 검색
- 학사 일정
- 스노웨이
- 수강신청 퀵링크
- 스노로즈 (교내 커뮤니티)
